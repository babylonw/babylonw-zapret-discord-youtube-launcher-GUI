# 🛡️ Zapret Launcher – графический управлятор для zapret

**Zapret Launcher** – это удобное приложение с графическим интерфейсом для управления утилитой **zapret** (обход DPI). Программа позволяет выбирать стратегии обхода, проверять их работоспособность, настраивать автозапуск и менять тему интерфейса. Всё в одном EXE-файле.

## 🚀 Быстрый старт

1. Запустите `ZapretLauncher.exe` **от имени администратора** (это необходимо для работы zapret).
2. Выберите стратегию обхода из выпадающего списка.
3. Нажмите **«Запустить обход»** – стратегия запустится в фоновом режиме без окон.
4. Чтобы проверить, работает ли выбранная стратегия, нажмите **«Проверить стратегию»** – программа выполнит HTTP/TLS-тесты и пинг основных сервисов (Discord, YouTube, Google, Cloudflare).
5. При необходимости включите **автозапуск** – стратегия будет стартовать вместе с Windows.
6. Для управления службами zapret используйте кнопку **«Запустить сервис»** (откроется `service.bat`).

## ⚙️ Что делает лаунчер

- Распаковывает все файлы zapret в папку `%LOCALAPPDATA%\ZapretBundle` (при первом запуске).
- Автоматически создаёт ярлык на рабочем столе.
- Запускает выбранный BAT-файл полностью скрыто (без мелькания окон).
- Позволяет проверить работоспособность стратегии через HTTP/TLS/пинг.
- Управляет автозапуском через планировщик задач Windows.
- Поддерживает светлую и тёмную темы.

## 🧪 Реакция антивирусов

Некоторые антивирусы (особенно Windows Defender, Avast, Kaspersky) могут ложно срабатывать на лаунчер и на `winws.exe`. Это связано с тем, что программа:

- Изменяет сетевые пакеты (обход DPI).
- Запускает процессы с правами администратора.
- Использует низкоуровневые сетевые операции (WinDivert).

**Важно:** программа **не является вредоносной**, не собирает данные, не содержит вирусов или троянов. Все исходные коды открыты. Если антивирус блокирует файл, добавьте его в исключения.
